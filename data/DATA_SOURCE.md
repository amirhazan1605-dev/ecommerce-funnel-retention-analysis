# Data

The raw dataset (~105 MB) is not stored in this repository due to its size and licensing.

It is the "eCommerce events history in electronics store" dataset by mkechinov on Kaggle:
https://www.kaggle.com/datasets/mkechinov/ecommerce-events-history-in-electronics-store

To reproduce this project, download the CSV from the link above and load it into PostgreSQL (see the SQL files and README for the table structure).
